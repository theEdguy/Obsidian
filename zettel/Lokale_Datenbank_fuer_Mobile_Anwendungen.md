# [[Lokale_Datenbank_fuer_Mobile_Anwendungen]]

- **Kernkonzept:** Eine **Datenbank** ist ein strukturiertes System zur persistenten Speicherung, Verwaltung und Abfrage von Daten. Im Kontext mobiler Anwendungen bezeichnet eine **lokale Datenbank** eine auf dem Endgerät (z. B. Smartphone) installierte Datenbanklösung, die ohne Netzwerkverbindung arbeitet und Datenschutzanforderungen durch dezentrale Speicherung erfüllt. Sie ermöglicht die effiziente Verwaltung von [[Datenmodell|Datenmodellen]] wie individualisierten Touren oder aggregierten Nutzerdaten, ohne auf externe [[Client-Server-Architektur|Server-Infrastrukturen]] angewiesen zu sein.
