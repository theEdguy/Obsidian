# [[Message Oriented Middleware (MOM)]]

- **Kernkonzept:** Middleware für asynchrone, persistente Kommunikation. Queue Manager verwalten Warteschlangen (Queues). Sender platzieren Nachrichten in lokalen Queues, von wo sie über unidirektionale Channels mittels Message Channel Agents (MCA) geroutet werden. Message Broker dienen zur Übersetzung heterogener Nachrichtenformate.
