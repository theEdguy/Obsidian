# [[Required_Interface]]

- **Kernkonzept:** Ein **Required_Interface** (auch *benötigte Schnittstelle* genannt) ist eine [[Schnittstelle]], die von einer Komponente (z. B. einer Klasse oder einem Modul) *verlangt* wird, um ihre Funktionalität zu erfüllen. Es definiert die Methoden oder Operationen, die eine andere Komponente bereitstellen muss, damit die abhängige Komponente korrekt arbeiten kann. Im Gegensatz zu einer *provided interface* (bereitgestellte Schnittstelle), die eine Komponente selbst implementiert, beschreibt das *Required_Interface* die Abhängigkeiten einer Komponente nach außen. Es ist ein zentrales Konzept im [[Dependency_Inversion_Principle]] und wird häufig in [[Komponentendiagramm|Komponenten-]] oder [[Klassendiagramm|Klassendiagrammen]] modelliert, um lose Kopplung zu fördern.
