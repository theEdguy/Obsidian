# [[Code-Migration (Modelle)]]

- **Kernkonzept:** Strategien zum Verschieben von Programmcode in verteilten Systemen. Client-Server (CS): Code verbleibt auf Server. Remote Evaluation (REV): Client sendet Code an Server, Server führt aus und liefert Ergebnis. Code-on-Demand (CoD): Client lädt Code von Server und führt ihn lokal aus. Mobile Agents (MA): Autonomer Prozess wandert samt Code und Zustand von Host zu Host.
