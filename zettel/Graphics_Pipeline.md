# [[Graphics Pipeline]]

- **Kernkonzept:** Der standardisierte Ablauf zur Darstellung von 3D-Geometrie auf einem 2D-Rasterbildschirm. Phasen: 1) Vertex Processing (Transformation von Punkten via Vertex Shader). 2) Rasterung (Primitive in Fragmente zerlegen). 3) Fragment Processing (Berechnung der Pixelfarbe via Fragment Shader). 4) Blending (Kombination mit dem Framebuffer unter Beachtung des z-Buffers).
