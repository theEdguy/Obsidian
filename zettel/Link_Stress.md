# [[Link Stress]]

- **Kernkonzept:** Metrik für die Effizienz von Multicast-Kommunikation. Gibt an, wie viele Kopien derselben Nachricht über einen spezifischen physischen Netzwerklink gesendet werden müssen. Ziel von Application-Level Multicast (ALM) ist ein Link Stress von 1 auf allen physischen Links.
